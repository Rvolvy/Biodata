Tentang:
Perkenalkan nama saya Pradipta Raffa Pratama, saya berumur 16 tahun dan saya adalah seorang pelajar tingkat SMK di jurusan RPL. Saya memiliki ketertarikan pada hal yang berkaitan dengan teknologi terkini, seperti design pada software Adobe Photoshop hingga menggambar digital pada software SketchBook
Selain itu, saya juga tertarik dengan pemrograman web dan pembuatan aplikasi. Saat beranjak masuk SMK saya mulai mencoba aplikasi yang bernama Mimo yang berisi tentang pemrograman web. Disitu saya mulai mempelajari dari html basic sampai style CSS.


Personal:
Saya sendiri memiliki hobi dalam bidang seni/design, gaming dan juga travelling. Saya juga selalu mencari hal baru yang bisa saya kerjakan untuk mengisi waktu luang seperti mencoba style gambar layaknya Manga, design logo pada software Photoshop ataupun Illustrator, dan mencoba memasukkan syntax baru yang saya baru pahami.
Selain itu, saya sering kali bermain game untuk mencari referensi objek, karakter, bahkan ceritanya.
Saya juga suka Travelling, sejak dulu saya seringkali berjelajah, entah itu memakai kendaraan ataupun jalan kaki. Saya selalu bosan untuk diam di rumah, tapi semenjak pandemi, saya sudah tidak sering seperti dulu, saya hanya bisa diam dan mencari kesibukan baru yang bisa dilakukan di rumah.
Saya juga memiliki cita-cita menjadi Creative Director di suatu perusahaan, dan saya juga ingin menjadi Freelancer di bidang designer logo.
